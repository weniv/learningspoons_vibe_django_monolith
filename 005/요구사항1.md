나는 Django를 이제 막 배우는 초보 개발자야. 강사님이 `명령어_백업.md` 파일을 이전시간에 작성하여 주었어.

이 파일을 참고해서 실제 동작하는 Django 웹 애플리케이션 지시사항 파일인 `설계/명령어.md` 파일을 만들고 싶어. 이 파일에 추가되어야 하는 사항은 다음과 같아.

1. 로그인, 로그아웃
2. 회원가입
3. 게시글 작성, 수정, 삭제
4. 게시글 목록 보기, 상세 보기
5. 댓글 작성, 수정, 삭제
6. 마크다운 지원

이러한 방식을 직접만들어볼 수 있도록 아래 파일을 순서대로 작성해줘. 설계 폴더에 각각 만들어야해. 우선은 1번부터 작성해줘. 순차적으로 내가 지시할게. 명령어를 설계할 때에는 CBV 방식으로 작성해줘.

1. `설계/API_설계.md`: 각 기능에 대한 API 설계 문서
2. `설계/모델_설계.md`: 데이터베이스 모델 설계 문서, ERD 머메이드로 표현
3. `설계/뷰_설계.md`: 각 기능에 대한 뷰 설계 문서
4. `설계/명령어.md`: 실제 Django 명령어 및 코드 작성 지시사항
